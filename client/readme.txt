This is a clone application of Chat-GPT

I have created the frontend of this website using pure HTML,CSS and JavaScript,
and created Backend with the help of Node.js and express, In order to get the response we had used an API from 
the OpenAi platform.
****************************
How To Use the Application

Just install all the Dependency and you are good to go, you just have to switch to the client folder and write npm run dev
for starting the Web-Application,
After the Application has been started now you have to create another terminal and switch to the server folder and write npm run server
When the server get started switch to respected PORT, 

Then just type your query
